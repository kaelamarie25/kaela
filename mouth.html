CSS Zen Mouth

front-end web development

The first project I was assigned during an advanced web design course I took in college was to design and develop a sort of 'microsite' around the parameters set by CSS Zen Garden, a free content resource designed to demonstrate what can be accomplished using only CSS. I tend to work very quickly, and this project was no exception. So when I finished my initial project in only 1 of the 3 weeks we were given to complete the project, I decided to make another one. And thus, CSS Zen Mouth was born!
